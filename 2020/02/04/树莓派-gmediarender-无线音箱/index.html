Tuesday, February 4th 2020

无线音箱，DLNA推送音频，gmediarender -f “raspberry pi”

基于gmediarender树莓派搭建DLNA客户端，只支持音频

目前搭建DLNA服务器有minidlna，不过不能进行DLNA推送，只能执行树莓派的媒体服务器

1 更新源，安装依赖

+ sudo apt-get update
+ sudo apt-get install libupnp-dev libgstreamer1.0-dev \gstreamer1.0-plugins-base gstreamer1.0-plugins-good \gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly \gstreamer1.0-alsa autoconf

2 安装gmediarender

git clone https://github.com/hzeller/gmrender-resurrect.git
cd gmrender-resurrect
sudo ./autogen.sh
sudo ./configure
sudo make 
sudo make install

3 启动gmediarender

sudo gmediarender -f "输入DLNA设备名称"

4 添加到启动项

sudo nano /etc/rc.local

在exit 0 前添加

sudo gmediarender -f "设备名称" &

&的意思是作为系统进程开机运行
